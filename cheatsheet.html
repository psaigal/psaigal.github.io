for...in in Ruby iterates through each item in an Array. In a hash for...in iterates through each key an value individually. 

  var number_list = [1, 2, 3, 4];

JavaScript's for...in iterates through an array by its index, whereas Ruby's for...in or each method iterates through each item in the array/object directly. for...in in JavaScript iterates through an object by each property.

each loop goes through each item in a list (like an array or hash) and executes the code inside the block 


  var each = function(array, iterate) {
    for (var i=0; i < array.length; i++) {
      iterate(array[i]);
    }
  };

  var desserts = ["ice cream", "brownies", "cookies"];

  each(desserts, function(element) { console.log("I want " + element + "!"); });
            

  >I want ice cream!
  >I want brownies!
  >I want cookies!
each function takes an array and goes through each element in the array using a nested function
